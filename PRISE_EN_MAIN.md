# Meet Notes — Guide de prise en main
**Ramery BTP · Outil de transcription et compte rendu automatisé**

---

## Accès

Ouvrez l'application dans votre navigateur à l'adresse fournie par votre administrateur.
Aucune installation requise.

---

## Comment ça marche ?

L'application traite un enregistrement audio en **4 étapes** guidées :

```
📁 Upload  →  🎙 Transcription  →  👥 Participants  →  📄 Rapport
```

---

## Étape 1 — Importer votre audio

Deux options :

- **Importer un fichier** : glissez-déposez ou cliquez pour sélectionner un fichier audio  
  Formats acceptés : `mp3` · `wav` · `m4a` · `ogg` · `flac` · `aac` · `webm`

- **Enregistrer en direct** : cliquez sur le bouton micro pour démarrer, recliquez pour arrêter

**Avant de lancer l'analyse, configurez dans la barre latérale gauche :**

| Paramètre | Description |
|-----------|-------------|
| **Langue** | Français ou Anglais |
| **Template** | Type de réunion (voir ci-dessous) |
| **Contexte additionnel** | Informations utiles pour le rapport (lieu, sujet précis…) |

---

## Choisir le bon template

| Template | Usage |
|----------|-------|
| 📋 Socle commun | Modèle standard Ramery — adapté à tous types de réunions BTP |
| ✏️ Custom | Partez du socle commun et personnalisez le prompt selon vos besoins |

> **Custom** : sélectionnez *Custom* dans la liste, modifiez le prompt affiché, donnez-lui un nom et cliquez sur **Enregistrer comme template**. Il sera disponible pour vos prochaines analyses.

---

## Étape 2 — Transcription

Cliquez sur **▶ Lancer l'analyse**.  
La transcription est effectuée automatiquement par l'IA (Gemini).  
Durée estimée : **1 à 3 minutes** selon la longueur de l'enregistrement.

> **Option** : cochez *"Transcription uniquement"* si vous ne souhaitez pas générer de rapport.

---

## Étape 3 — Identifier les participants

L'IA détecte automatiquement les différents locuteurs.  
Vous pouvez renseigner les vrais prénoms/noms dans le tableau pour personnaliser le rapport.  
Cette étape est **optionnelle**.

---

## Étape 4 — Rapport et export

Le rapport est affiché directement dans l'application.  
Téléchargez-le dans le format de votre choix :

| Format | Usage recommandé |
|--------|-----------------|
| ⬇ `.pdf` | Envoi par email, archivage |
| ⬇ `.docx` | Modification dans Word |
| ⬇ `.md` | Usage technique |

Le transcript complet est également disponible dans l'onglet **Transcript**.

---

## Commencer une nouvelle analyse

Cliquez sur **↩ Nouvelle analyse** dans la barre latérale pour revenir à l'étape 1.
